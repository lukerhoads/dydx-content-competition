# DYDX Risk Calculator

Risk calculator for https://dydx.exchange built in React.

## Details
- Font: Inter
- Background color: #1c1c28
- Purple color: #6669ff

# How to use
- Clone
- Yarn/npm install
- Yarn/npm start
- view localhost:3000
